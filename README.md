# CoronaKit
This is a toolkit to embed the status of COVID-19 in your python projects


## Usage

Just import CoronaKit and do CoronaKit.**Status** (*Status could be Cases, Deaths, or Recovered*)

(Make sure you have CoronaKit in the same folder as your project!)

### Example 1

import CoronaKit

print(CoronaKit.Cases)

### Example 2

import CoronaKit

print("Current COVID-19 deaths: " + str(CoronaKit.Deaths))

### Example 3

from CoronaKit import *

print("People recovered from COVID-19: " + str(Recovered))
